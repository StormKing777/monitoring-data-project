# Observability Suite: Metrics & Monitoring

## Project Summary
This project is about **Monitoring**. I used **Prometheus** to collect data from my servers and **Grafana** to turn that data into beautiful charts and graphs so I can see if the system is happy or sad.

* **Languages Used:** SQL (for data queries).
* **Environments Used:** Linux, Kubernetes.
* **Technologies Used:** Prometheus, Grafana, MySQL.
